---
title: Menubar
description: Displays an 8-bit menubar component.
---

## Installation

```bash
npx shadcn@latest add @8bitcn/menubar
```

## Usage

```tsx showLineNumbers

  Menubar,
  MenubarContent,
  MenubarItem,
  MenubarMenu,
  MenubarSeparator,
  MenubarShortcut,
  MenubarTrigger,
} from "@/components/ui/8bit/menubar"
```
